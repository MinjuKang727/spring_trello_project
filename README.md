# 🏃내일lo
> 내일lo는 프로젝트 관리를 위한 협업 툴로, 직관적인 **칸반 보드(Kanban Board)** 방식으로 작업을 관리할 수 있는 도구입니다.

## 개발 Scope 및 담당 팀원
- **필수 기능 구현**
    - 회원가입/로그인 : `이봉원`
        - Spring Security
        - OAuth(카카오 로그인)
        - Redis
        - 로그아웃
    - 멤버 및 역할 관리 : `이지택`
    - 워크스페이스(Workspace) : `이지택`
    - 보드(Board) : `전우성`
    - 덱(Deck) : `강민주`
    - 카드(Card) : `신승재`
    - 댓글(Comment) : `전우성`
    - 첨부파일(Attachment) : `신승재`
    - 알림(Notification) : `강민주`
      - 워크스페이스별 private 채널 생성
      - 워크스페이스 멤버 초대
      - 워크스페이스 변경 사항 알람 메세지 전송
    - 검색
        - 카드 단/다건 조회 : `신승재`
        - 보드 단/다건 조회 : `전우성`
        - 워크스페이스 단건 조회 : `이지택`
- **도전 기능**
    - 최적화(Indexing)
    - 캐싱(Caching) : `신승재`
